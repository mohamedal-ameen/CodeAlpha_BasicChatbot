# 🤖 CodeAlpha Basic Chatbot

A simple rule-based chatbot built using Python as part of the CodeAlpha Python Programming Internship.

## 🚀 Features

- Responds to basic greetings
- Answers predefined questions
- Console-based chatbot
- Beginner-friendly

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Install Python.
2. Open the project folder.
3. Run:

```bash
python chatbot.py
```
